# Картотека граждан

Frontend-демо учёта граждан: dashboard со сводкой, картотека со списком и фильтрами, карточка с разделами и полями ввода.

Стек: React, TypeScript, Vite, MUI, Recharts, React Router. Данные моковые, без бэкенда.

## Экраны

- **Главная** (`/`) — KPI и графики по возрасту, статусу учёта и регионам
- **Картотека** (`/citizens`) — таблица граждан, поиск по ФИО, фильтры по региону и статусу, пагинация, счётчик записей
- **Карточка** (`/citizens/:id`) — разделы: основные сведения, семья, образование, труд и соцстатус, документы; сохранение в mock-store

## Запуск

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
npm run preview
```

## Данные

В памяти генерируется около 136 тысяч записей граждан со связанными списками (семья, образование, трудовая история, документы). На главной сводка считается по всему объёму — так показан масштаб из ТЗ (100 000+ записей). Список картотеки отдаётся постранично.
